Basic HTML Elements

HTML uses different elements to structure content on a webpage. These basic tags are essential in creating a simple and functional website.

Headings (h1 – h6)

Heading tags are used for titles and subtitles. <h1> is the largest heading while <h6> is the smallest.

Paragraph (<p>)

The paragraph tag is used to display blocks of text on a webpage.

Images (<img>)

The image tag displays pictures on a webpage. It requires a source attribute and does not need a closing tag.

Line Break / Space (<br>)

The line break tag creates a new line inside text without starting a new paragraph.

Division (<div>)

The div tag is a container used to group elements together. It helps organize page layout and styling.

Navigation (<nav>)

The nav tag contains navigation links that connect different pages of the website.

Span (<span>)

The span tag is an inline container used to style or modify small portions of text.

Lists

HTML lists organize content. Unordered lists use <ul>, ordered lists use <ol>, and list items use <li>.

These essential HTML elements work together to create a simple, organized, and readable webpage.
